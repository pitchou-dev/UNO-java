UNO Console Game (Java)


Project Overview:

This project is a console-based implementation of the UNO card game, developed in Java. The game follows the basic rules of UNO and allows players to play directly from the terminal without any graphical interface.

The goal of this project is to practice object-oriented programming (OOP) concepts such as classes, inheritance, encapsulation, and game logic management in Java.


The Project's structure:
Menu.java
Deck.java
Game.java
Actionable.java
Actioncard.java
Actiontype.java
NumberCard.java
Wildcard.java
Wildtype.java
Demo.java
Player.java
Botplayer.java



How to Run the Project

1-Make sure you have Java JDK installed on your machine.

2-Open the project in your IDE or terminal.

3-The main entry point of the game is located in the Menu class.

4-Run the Menu class to start the game.
